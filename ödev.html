What is AngularJS?

Angular JS is a structural framework for dynamic web applications. Although it allows to use HTML as a template language,
one of its biggest advantages is that it is supported by google. For this reason, it is widely used. Compared to other JS libraries, it enables much more work to be done thanks to the codes written shorter.
Angular eliminates most of the code without having to write bulk code,
making it an ideal partner to work with almost any server technology found in most browsers.



How To Install AngularJS?

First we need to install NodeJS. We need to download and install Node.js from https://nodejs.org/en/.After the Node.js installation is finished, we need to open the cmd screen and install Angular via npm (Node Package Manager). Doing this is as simple as below. npm install -g @angular/cli we write the command and run it. We are performing the angular/cli installation via NPM. Where -g is written to ensure that it loads globally. After the installation process has been completed, we can now start developing our Angular applications.

What Is The NG-APP Directive?

NG-APP Directive The ng-app directive is captured by the HTML element of the implementation of the Angularjs implementation. This directive will automatically run the application when the application page finishes loading.




What Is The NG-INIT Directive?

NG-INIT Directive The ng-init directive allows us to define an initial value for an expression for the current carrier scope, not much use for page readability reduction in terms of MVC structure.

NG-BIND Directive With the ng-bind directfit you can link any element in scope to any element on the DOM. With this mechanic, which we call data binding, changes to the variable will be instantaneously applied to the DOM element content as well.



What Is The NG-MODEL Directive?

NgModeles configure the injector and the compiler and help organize related things together. Ng-Modele takes a metadata object that describes how to compile a component's template and how to create an injector at runtime. It identifies the modele's own components, directives, and pipes, making some of them public, through the exports property, so that external components can use them. Ng-Modele can also add service providers to the application dependency injectors.



What Is The NG-Value Directive?

Binds the given expression to the value of the element. It is mainly used on input[radio] and option elements, so that when the element is selected, the ngModel of that element (or its select parent element) is set to the bound value. It is especially useful for dynamically generated lists using ngRepeat, as shown below. It can also be used to achieve one-way binding of a given expression to an input element such as an input[text] or a textarea, when that element does not use ngModel.
